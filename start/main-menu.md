# 메인 메뉴

Eartopia 메인 메뉴는 자주 쓰는 기능을 한곳에서 열 수 있는 개인 메뉴예요. 낚시, 언어, 꾸미기 같은 기능으로 빠르게 이동할 수 있어요.

## 무엇을 하면 되나요?

- 기본 설정에서는 웅크리기 상태로 손 바꾸기 키를 눌러요. 보통 `Shift+F`예요.
- 나침반을 들고 우클릭해도 메인 메뉴를 열 수 있어요.
- 메뉴 안에서는 낚시 패널, 낚시 통계, 낚시 상점, 꾸미기, 언어 설정으로 이동할 수 있어요.

## 명령어

| 명령어 | 용도 |
| --- | --- |
| `/eartopiamainmenu` | 메인 메뉴를 열어요. |
| `/emainmenu` | 메인 메뉴의 짧은 별칭이에요. |

## 알아두면 좋아요

- 메인 메뉴는 본인에게만 보이는 개인 패널이에요.
- 메뉴가 보이지 않으면 리소스팩 적용 상태를 먼저 확인해요.
- 키 조합이 동작하지 않으면 나침반 우클릭이나 명령어로 열어봐요.
