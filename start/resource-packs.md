# 리소스팩

Eartopia의 커스텀 아이템, 낚시 UI, 농작물 모델, 꾸미기 스킨, 패널 화면은 리소스팩을 통해 보여요. 안내가 뜨면 수락해두는 걸 추천해요.

## 무엇을 하면 되나요?

1. 접속할 때 리소스팩 적용 안내가 보이면 수락해요.
2. 리소스팩이 바뀌었다는 안내가 오면 `/resourcepacks update`를 눌러 적용해요.
3. 제대로 적용됐는지 궁금하면 `/resourcepacks status`로 확인해요.
4. 적용 목록이 궁금하면 `/resourcepacks list`를 사용해요.

## 명령어

| 명령어 | 용도 |
| --- | --- |
| `/resourcepacks update` | 대기 중인 리소스팩 업데이트를 적용해요. |
| `/resourcepacks status` | 내 리소스팩 상태를 확인해요. |
| `/resourcepacks list` | 서버가 관리하는 리소스팩 목록을 봐요. |
| `/resourcepacks gui` | 리소스팩 관련 GUI를 열어요. |
| `/packs` | `/resourcepacks`의 짧은 별칭이에요. |
| `/erpacks` | `/resourcepacks`의 별칭이에요. |
