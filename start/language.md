# 언어 설정

Eartopia는 여러 언어로 플레이어 메시지를 보여주는 언어 시스템을 사용해요. 직접 언어를 고르거나, 마인크래프트 클라이언트 언어를 따라가도록 둘 수 있어요.

## 무엇을 하면 되나요?

- 현재 언어가 맞는지 `/language current`로 확인해요.
- 한국어로 고정하고 싶으면 `/language set ko_KR`을 사용해요.
- 자동 감지를 다시 쓰고 싶으면 `/language auto`를 사용해요.
- 사용할 수 있는 언어 목록은 `/lang`으로 확인해요.

## 명령어

| 명령어 | 용도 |
| --- | --- |
| `/language current` | 지금 적용된 언어를 확인해요. |
| `/language list` | 선택할 수 있는 언어 목록을 봐요. |
| `/language auto` | 수동 선택을 해제하고 자동 감지를 사용해요. |
| `/language set ko_KR` | 한국어로 설정해요. |
| `/lang` | `/language`와 같은 언어 명령어예요. |
| `/locale` | `/language`와 같은 언어 명령어예요. |

## 지원 언어

- `ko_KR`: 한국어
- `en_US`: 영어
- `ja_JP`: 일본어
- `zh_CN`: 중국어 간체
- `zh_TW`: 중국어 번체
- `vi_VN`: 베트남어

## 알아두면 좋아요

- 일부 기능은 아이템 이름과 설명도 선택한 언어에 맞춰 보여줘요.
- 지원하지 않는 언어가 감지되면 영어 표시가 기본 fallback으로 사용돼요.
- 언어를 바꾼 뒤에도 오래된 화면이 남아 보이면 메뉴를 다시 열어봐요.
