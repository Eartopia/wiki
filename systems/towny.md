# 마을

Eartopia의 마을 시스템은 Towny를 기반으로 해요. 플레이어는 주민, 마을, 국가, 플롯 같은 단위로 땅과 커뮤니티를 관리할 수 있어요.

## 무엇을 하면 되나요?

- 마을 기능이 처음이라면 `/towny`와 `/town ?`로 도움말을 확인해요.
- 내 주민 정보는 `/resident` 또는 `/res`로 확인해요.
- 마을에 들어가거나 새 마을을 만들 때는 서버 안의 안내와 마을장 안내를 함께 확인해요.
- 플롯 단위로 땅을 관리할 때는 `/plot ?`를 참고해요.

## 명령어

| 명령어 | 용도 |
| --- | --- |
| `/towny` | Towny 전체 도움말과 정보를 확인해요. |
| `/town` 또는 `/t` | 마을 관련 명령어를 사용해요. |
| `/nation` 또는 `/n` | 국가 관련 명령어를 사용해요. |
| `/plot` 또는 `/p` | 플롯 관련 명령어를 사용해요. |
| `/resident` 또는 `/res` | 내 주민 정보를 확인해요. |
| `/invite` | 초대 관련 명령어를 사용해요. |

## 알아두면 좋아요

- Towny는 마을, 국가, 플롯, 보호, 경제 기능을 함께 다루는 큰 시스템이에요.
- 마을별 규칙과 건축 방향은 마을마다 다를 수 있어요.
- 다른 사람의 마을이나 플롯에서는 권한이 없으면 건축, 파괴, 상호작용이 제한될 수 있어요.
- 자세한 하위 명령어는 게임 안에서 각 명령어 뒤에 `?`를 붙여 확인해요.
