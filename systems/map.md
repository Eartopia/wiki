# 지도

Eartopia는 웹에서 볼 수 있는 공개 지도를 제공해요. 월드의 지형, 위치, 마커, 플레이어 표시를 확인할 때 사용해요.

## 무엇을 하면 되나요?

- 브라우저에서 [https://map.eartopia.net/](https://map.eartopia.net/)을 열어요.
- 2D 지도와 3D 지구본 보기를 전환하며 위치를 살펴봐요.
- 마을이나 이동 경로를 찾을 때 지도와 게임 안 좌표를 함께 확인해요.

## 명령어

| 명령어 | 용도 |
| --- | --- |
| `/dynmap` | 지도 관련 기본 안내를 확인해요. |

## 알아두면 좋아요

- 공개 지도는 게임 안 위치를 파악하는 보조 도구예요.
- 지도 페이지가 잠시 열리지 않으면 잠시 뒤 다시 시도해요.
