# 문제 해결

문제가 생겼을 때 먼저 확인할 수 있는 기본 점검 목록이에요. 서버 정책이나 운영 문의는 공식 안내가 준비되면 별도 페이지로 정리해요.

## 리소스팩이 안 보여요

1. `/resourcepacks status`로 현재 상태를 확인해요.
2. 업데이트 안내가 있었다면 `/resourcepacks update`를 사용해요.
3. 마인크래프트 서버 목록에서 Eartopia의 서버 리소스팩 설정이 켜져 있는지 확인해요.
4. 그래도 모델이 기본 아이템처럼 보이면 재접속해봐요.

## 언어가 원하는 대로 안 나와요

1. `/language current`로 현재 언어를 확인해요.
2. 한국어로 고정하려면 `/language set ko_KR`을 사용해요.
3. 자동 감지를 쓰려면 `/language auto`를 사용해요.
4. 메뉴나 아이템 이름이 그대로라면 화면을 다시 열어봐요.

## 메인 메뉴가 안 열려요

1. `Shift+F`를 다시 시도해요.
2. 나침반을 들고 우클릭해봐요.
3. `/eartopiamainmenu` 또는 `/emainmenu`로 열어봐요.
4. 메뉴 화면이 이상하면 리소스팩 상태를 확인해요.

## 패널 커서가 모서리에서 어긋나요

1. `/panels calibrate`로 커서 보정을 시작해요.
2. 검은 패널에 표시되는 빨간 점을 화면 중앙 조준점으로 정확히 클릭해요.
3. 안내되는 순서대로 가운데와 네 모서리를 클릭해요.
4. 기본 4x4 패널이 너무 크게 느껴지면 `/panels calibrate 3`으로 다시 보정해요.
5. 중간에 멈추고 싶다면 `/panels calibrate cancel`로 보정 패널을 닫아요.
6. 보정 후 더 불편해졌다면 `/panels calibrate reset`으로 기본값으로 되돌려요.

## 지도가 안 열려요

1. [https://map.eartopia.net/](https://map.eartopia.net/)을 다시 열어봐요.
2. 브라우저 새로고침을 해요.
3. 잠시 접속이 불안정할 수 있으니 시간을 두고 다시 시도해요.

## 알아두면 좋아요

- 같은 문제가 반복되면 어떤 명령어를 썼는지, 어떤 화면에서 멈췄는지 기록해두면 좋아요.
