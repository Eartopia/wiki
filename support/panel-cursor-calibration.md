# 패널 커서 보정

패널 화면에서 가운데는 잘 맞지만 모서리 쪽으로 갈수록 커서가 조금 어긋나 보일 수 있어요. 이럴 때는 패널 커서 보정을 진행하면 내 화면 기준에 맞게 커서 위치가 더 정확해져요.

## 보정 시작하기

`/panels calibrate`를 입력하면 검은 보정 패널이 열려요.

보정 중에는 패널 안의 커서 표시가 잠시 꺼져요. 화면 중앙의 조준점으로 패널에 표시된 십자가를 정확히 바라보고 클릭해 주세요.

클릭 순서는 다음과 같아요.

1. 가운데
2. 왼쪽 위
3. 오른쪽 위
4. 오른쪽 아래
5. 왼쪽 아래

모든 지점을 정상적으로 클릭하면 보정값이 저장되고, 이후 패널을 사용할 때 그 보정값이 적용돼요.

## 더 큰 패널로 보정하기

기본 보정으로도 모서리 오차가 남는다면 `/panels calibrate 4`를 사용할 수 있어요. 더 큰 4x4 보정 패널이 열려서 넓은 화면 기준으로 다시 맞출 수 있어요.

## 보정 되돌리기

언제든지 `/panels calibrate reset`을 입력하면 기본값으로 돌아가요.

보정 후 오히려 커서가 더 어긋나거나, 다른 화면 비율에서 불편하게 느껴진다면 reset으로 되돌린 뒤 다시 보정해보세요.

## 클릭이 거부될 때

십자가에서 너무 먼 곳을 클릭하면 해당 클릭은 저장되지 않아요. 패널이 다시 같은 지점을 안내하면, 조준점을 십자가 중앙에 맞추고 다시 클릭해 주세요.

잘못된 클릭으로 패널을 사용할 수 없게 되는 일을 막기 위해, 너무 이상한 보정값은 자동으로 거부돼요.
