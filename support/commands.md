# 명령어 모음

이 페이지는 Eartopia Wiki v1에서 확인된 플레이어용 명령어를 모아둔 곳이에요. 관리자용 명령어와 운영자 전용 기능은 제외해요.

## 시작과 설정

| 명령어 | 용도 |
| --- | --- |
| `/spawn` | 스폰으로 이동해요. |
| `/스폰` | 스폰으로 이동해요. |
| `/넴주` | 스폰으로 이동해요. |
| `/language current` | 현재 언어를 확인해요. |
| `/language list` | 지원 언어 목록을 봐요. |
| `/language auto` | 언어 자동 감지를 사용해요. |
| `/language set ko_KR` | 한국어로 설정해요. |
| `/lang` | 언어 명령어 별칭이에요. |
| `/locale` | 언어 명령어 별칭이에요. |
| `/resourcepacks update` | 리소스팩 업데이트를 적용해요. |
| `/resourcepacks status` | 리소스팩 상태를 확인해요. |
| `/resourcepacks list` | 리소스팩 목록을 확인해요. |
| `/packs` | 리소스팩 명령어 별칭이에요. |
| `/eartopiamainmenu` | 메인 메뉴를 열어요. |
| `/emainmenu` | 메인 메뉴 명령어 별칭이에요. |

## 마을과 지도

| 명령어 | 용도 |
| --- | --- |
| `/towny` | Towny 도움말과 정보를 확인해요. |
| `/town` 또는 `/t` | 마을 명령어를 사용해요. |
| `/nation` 또는 `/n` | 국가 명령어를 사용해요. |
| `/plot` 또는 `/p` | 플롯 명령어를 사용해요. |
| `/resident` 또는 `/res` | 주민 정보를 확인해요. |
| `/invite` | 초대 관련 명령어를 사용해요. |
| `/dynmap` | 지도 관련 기본 안내를 확인해요. |

## 생활 콘텐츠

| 명령어 | 용도 |
| --- | --- |
| `/fish` | 낚시 메뉴를 열어요. |
| `/fish panel` | 패널형 낚시 화면을 열어요. |
| `/fish classic` | 인벤토리형 낚시 메뉴를 열어요. |
| `/fish gear` | 낚시 장비를 확인해요. |
| `/fish shop` | 낚시 상점을 열어요. |
| `/fish index` | 물고기 도감을 확인해요. |
| `/fish stats` | 낚시 통계를 확인해요. |
| `/fish sell hand` | 손에 든 물고기를 팔아요. |
| `/fish sell all` | 판매 가능한 물고기를 모두 팔아요. |
| `/fish theme list` | 낚시 메뉴 테마 목록을 봐요. |
| `/fish theme <테마> [화면]` | 낚시 메뉴 테마를 바꾸고 화면을 열어요. |
| `/art` 또는 `/artmap` | 아트 메뉴를 열어요. |
| `/cosmetics` | 꾸미기 메뉴를 열어요. |
| `/skin` 또는 `/skins` | 꾸미기 메뉴 별칭이에요. |

## 알아두면 좋아요

- 각 시스템의 자세한 하위 명령어는 해당 기능 페이지를 확인해요.
- 명령어 권한은 서버 설정과 플레이 상태에 따라 달라질 수 있어요.
- 농사는 일반 플레이어용 명령어 없이 심기, 뼛가루, 수확 행동으로 진행해요.
