# 뉴비 필독

Eartopia를 처음 시작했다면 이 페이지부터 보면 좋아요. 서버에 들어온 뒤 바로 필요한 설정과 기본 이동, 메뉴, 콘텐츠를 한 번에 확인할 수 있어요.

## 무엇을 하면 되나요?

1. 서버 리소스팩 안내가 보이면 수락해요.
2. 채팅에 한국어가 자연스럽게 보이는지 확인해요.
3. `/language current`로 현재 언어를 확인해요.
4. `/spawn`으로 스폰 이동을 익혀요.
5. `Shift+F` 또는 나침반 우클릭으로 메인 메뉴를 열어봐요.
6. `/fish`, `/town`, `/resourcepacks status` 같은 기본 명령어를 천천히 익혀요.

공식 접속 주소와 권장 접속 버전은 아직 Wiki에 공개할 확인 근거가 없어요. 이미 안내받은 주소가 있다면 그 주소로 접속해요.

## 명령어

| 명령어 | 용도 |
| --- | --- |
| `/spawn` | 스폰으로 이동해요. |
| `/스폰` | 한국어 스폰 이동 명령어예요. |
| `/language` | 언어 설정을 열거나 확인해요. |
| `/resourcepacks status` | 리소스팩 적용 상태를 확인해요. |
| `/fish` | Eartopia Fishing 메뉴를 열어요. |
| `/town` | 마을 기능을 확인해요. |

## 알아두면 좋아요

- Eartopia의 일부 아이템, UI, 낚시 화면, 꾸미기 요소는 리소스팩이 있어야 제대로 보여요.
- 언어 설정은 여러 Eartopia 기능의 메시지와 아이템 이름에 영향을 줘요.
- 스폰 월드는 보호되어 있어요. 일반 플레이어는 스폰을 안전하게 둘러보는 용도로 사용해요.
- 공식 규칙과 운영 정책은 [이용 안내](rules.md)에 있어요.
