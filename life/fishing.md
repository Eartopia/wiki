# 낚시

Eartopia Fishing은 일반 낚시를 커스텀 미니게임, 장비 성장, 물고기 도감, 상점, 판매 시스템으로 확장한 생활 콘텐츠예요. 현재 구현 기준으로 물고기 60종, 낚싯대 12종, 낚싯줄 10종, 찌 10종, 마법 12종이 준비되어 있어요.

## 무엇을 하면 되나요?

1. `/fish`로 낚시 메뉴를 열어요.
2. `/fish gear`에서 장비를 확인해요.
3. 낚싯대를 던지고 입질 표시가 뜨면 다시 감아 미니게임을 시작해요.
4. 흰 영역 안에 물고기를 유지해서 진행도를 끝까지 채워요.
5. 잡은 물고기는 `/fish sell hand` 또는 `/fish sell all`로 판매해요.
6. 장비를 구매하고 장착하면서 더 무겁고 희귀한 물고기에 도전해요.
7. `/fish index`와 `/fish stats`로 진행도를 확인해요.

## 명령어

| 명령어 | 용도 |
| --- | --- |
| `/fish` | 낚시 기본 메뉴를 열어요. |
| `/fish panel` | 패널형 낚시 화면을 열어요. |
| `/fish classic` | 인벤토리형 낚시 메뉴를 열어요. |
| `/fish gear` | 낚시 장비를 확인해요. |
| `/fish shop` | 낚시 상점을 열어요. |
| `/fish index` | 물고기 도감을 확인해요. |
| `/fish stats` | 내 낚시 통계를 확인해요. |
| `/fish sell hand` | 손에 든 물고기를 팔아요. |
| `/fish sell all` | 판매 가능한 물고기를 한 번에 팔아요. |

## 미니게임 흐름

낚싯바늘이 물에 안정적으로 떠 있으면 Eartopia Fishing이 낚시를 맡아요. 물 밖에 있거나 엔티티를 낚은 상황은 일반 낚시 흐름을 방해하지 않아요.

1. 입질까지는 기본 15~30초 정도 걸려요.
2. 입질 대기 시간이 끝나면 물고기가 걸렸다는 액션바와 소리, 찌 근처 표시가 나와요.
3. 6초 안에 다시 감지 않으면 물고기가 빠져나가요.
4. 미니게임에서는 입력할 때마다 흰 조작 영역이 위로 움직이고, 중력과 감속 때문에 다시 내려와요.
5. 물고기의 중심을 흰 영역 안에 두면 진행도가 오르고, 벗어나면 진행도가 내려가요.
6. 한 번도 놓치지 않고 성공하면 퍼펙트 낚시로 처리돼요.

## 장비

장비는 낚싯대, 낚싯줄, 찌, 마법 네 부위로 나뉘어요. `/fish shop`에서 구매하고 `/fish gear`에서 장착해요.

| 부위 | 개수 | 시작 장비 | 최고 가격 장비 |
| --- | ---: | --- | --- |
| 낚싯대 | 12개 | 초보자 낚싯대 | 태양벼림 낚싯대 |
| 낚싯줄 | 10개 | 면 낚싯줄 | 태양선 낚싯줄 |
| 찌 | 10개 | 코르크 찌 | 태양 찌 |
| 마법 | 12개 | 마법 없음 | 명품 |

장비 스탯은 아래처럼 작동해요.

- 행운은 더 희귀한 물고기와 변이 확률에 영향을 줘요.
- 힘은 미니게임 손실과 물고기 속도 압박을 줄이는 쪽으로 작동해요.
- 숙련은 조작 영역과 진행도 유지에 도움을 줘요.
- 입질은 입질 대기 시간을 줄여요.
- 거물 확률은 더 큰 무게와 일부 변이 판정에 영향을 줘요.
- 최대 무게 제한이 낮으면 너무 무거운 물고기는 후보에서 빠질 수 있어요.

## 물고기와 희귀도

도감에는 현재 활성 물고기 60종이 있어요. 물고기는 지역 물 타입과 시간, 날씨, 장비 조건에 따라 후보가 달라져요.

| 희귀도 | 개수 | 예시 |
| --- | ---: | --- |
| 잡동사니 | 1종 | 떠내려온 나무 |
| 풍부 | 14종 | 피라미, 멸치, 정어리, 빙어 |
| 일반 | 15종 | 잉어, 무지개송어, 큰입배스, 대구 |
| 특이 | 12종 | 참돔, 노던파이크, 홍연어, 방어 |
| 은밀 | 9종 | 복어, 철갑상어, 왕연어, 피라루쿠 |
| 전설 | 3종 | 별빛 비단잉어, 주름상어, 실러캔스 |
| 신화 | 2종 | 심연 참치, 산갈치 |
| 유물 | 1종 | 고대 비늘 |
| 이국 | 1종 | 오로라 가오리 |
| 시크릿 | 1종 | 공허지느러미 |
| 울트라 시크릿 | 1종 | 가라앉은 왕관고기 |

## 변이와 보상

물고기는 기본 상태 외에 변이가 붙을 수 있어요.

| 변이 | 가치 배율 |
| --- | ---: |
| 반짝이는 | 1.8배 |
| 황금 | 3배 |
| 별빛 | 8배 |

잡은 물고기의 가치는 기본 가치, 무게 비율, 변이 배율을 함께 사용해 계산해요. 퍼펙트 낚시에 성공하면 가치와 경험치가 추가로 올라가요.

## 도감과 판매

- `/fish index`는 발견한 물고기 수와 전체 도감 수를 보여줘요.
- 새 물고기를 처음 잡으면 도감 진행도 액션바가 한 번 더 나와요.
- `/fish stats`는 총 획득 수, 퍼펙트 횟수, 최고 판매 가치를 보여줘요.
- `/fish sell hand`는 손에 든 판매 가능한 물고기를 팔아요.
- `/fish sell all`은 인벤토리 안의 판매 가능한 물고기를 모두 팔아요.

## 알아두면 좋아요

- 낚시 UI와 아이템 모델은 리소스팩이 있어야 제대로 보여요.
- `/fish`는 기본적으로 패널형 메인 화면을 열고, `/fish classic`은 인벤토리형 메뉴를 열어요.
