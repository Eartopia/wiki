# 환영해요

Eartopia Wiki에 오신 걸 환영해요.

이곳은 Eartopia 서버를 처음 시작하는 플레이어가 필요한 내용을 빠르게 찾을 수 있도록 정리한 한국어 안내서예요. 복잡한 운영 정보보다 실제 플레이에 바로 쓰는 메뉴, 명령어, 리소스팩, 마을, 낚시, 지도 안내를 먼저 다뤄요.

## 처음 왔다면

처음 접속했다면 아래 순서대로 확인해요.

1. [뉴비 필독](info/new-player.md)을 읽고 첫 설정을 확인해요.
2. 리소스팩 안내가 보이면 수락해요.
3. `/language` 또는 `/lang`으로 표시 언어를 확인해요.
4. `/spawn`, `/fish`, `/resourcepacks status` 같은 기본 명령어를 익혀요.
5. [지도](systems/map.md)에서 월드와 주요 위치를 둘러봐요.

## 주요 안내

- [빠른 시작](info/new-player.md): 처음 접속한 뒤 바로 해야 할 일을 정리해요.
- [언어 설정](start/language.md): 한국어, 영어, 일본어, 중국어, 베트남어 표시를 바꿔요.
- [리소스팩](start/resource-packs.md): Eartopia 전용 아이템과 UI가 제대로 보이도록 설정해요.
- [메인 메뉴](start/main-menu.md): `Shift+F` 또는 나침반으로 주요 기능을 열어요.
- [낚시](life/fishing.md): 커스텀 낚시, 장비, 상점, 도감 사용법을 확인해요.
- [마을](systems/towny.md): Towny 기반 마을, 국가, 플롯 기능을 시작해요.

## 준비 중인 정보

공식 접속 주소, 디스코드, 후원, 세부 이용 수칙, 처벌 정책은 아직 이 Wiki에 공개할 확인 근거가 없어요. 확인된 자료가 준비되면 별도 페이지로 정리해요.
